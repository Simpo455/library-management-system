
# Library Management System

WEX 328 Project 1 Submission

Tech Stack:
- Python FastAPI
- SQLAlchemy
- SQLite/PostgreSQL compatible
- Docker & Docker Compose

Features:
- User registration/login
- Role-based access (Admin, Librarian, Student)
- Book CRUD
- Borrow/Return books
- Reservations
- Reports API
- Dataset import support
